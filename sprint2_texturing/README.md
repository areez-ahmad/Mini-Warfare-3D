# Sprint 2: Texturing

## Description
UV unwrapping and texture mapping — applying materials, colors, and surface details to 3D models.

## Tasks
- Apply 2D image maps to 3D surfaces to add color, detail, and realism.
- UV unwrapping (flattening 3D surface to 2D).
- Creating diffuse/albedo maps, normal maps, and specular/roughness maps.
- Deliverables include textured models with UV maps and texture files (.png/.jpg/.tga).
